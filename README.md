# Agent Skills

Collection of local skills for AI coding agents.

## Available Skills

- `baseline-ui` - UI baseline constraints for Tailwind projects.
- `fixing-accessibility` - Targeted accessibility fixes for interactive UI.
- `fixing-metadata` - SEO and metadata audit/fix guidelines.
- `fixing-motion-performance` - Animation and rendering performance guardrails.
- `go-style-guide` - Go correctness, concurrency, API, and style best practices.
- `interface-design` - Interface product design process and quality bar.
- `numscript-guidelines` - Numscript DSL guidance for financial modeling.
- `react-patterns` - React 19 patterns, hooks, Server Components, and Actions.
- `tailwind-css-patterns` - Tailwind utility patterns and responsive design.
- `wcag-audit-patterns` - WCAG 2.2 audit and remediation checklist.

Each skill is defined in `skills/<skill-name>/SKILL.md`.
